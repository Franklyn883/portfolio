Project Description

WebSphere is a versatile blogging platform designed to allow users to create, edit, and delete blog posts, comments, and replies. The platform also supports liking and bookmarking posts, comments, and replies. The goal is to provide a smooth user experience with dynamic interactions using HTMX.

Key Features

  • User authentication with social signups
  • Blog post CRUD operations
  • Comment and reply management
  • Like and bookmark functionality
  • Responsive design
  • Interactive UI using HTMX

Technologies Used

HTML, CSS, JavaScript, Django, MySQL, e.t.c

Challenges

During the development of WebSphere, I faced a significant challenge when my laptop broke down. This left me without a laptop for about five months, resulting in a substantial gap in the development process. Despite this setback, I remained committed to the project and am currently working diligently to complete and deploy it as soon as possible.
